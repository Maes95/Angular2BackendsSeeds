# Angular2 + Node.js Backend

## Prerequisites

* node v7.6.0
* npm v4.1.2
* tsc v2.2.1

## SetUp

```sh
  git clone https://github.com/Maes95/Angular2BackendsSeeds.git
  cd Angular2+Node
  npm install
  cd frontend
  tsc
```

## Run
In Angular2+Node dolder

```sh
  npm start
```

Then, open your browser at http://localhost:8080/
