# Angular2 + Spring Backend

## Prerequisites

* Java 8
* Maven 3.0.5
* node 6.9.1
* npm 4.1.2
* tsc 2.2.1

## SetUp

```sh
  $ git clone https://github.com/Maes95/Angular2BackendsSeeds.git
  $ cd Angular2+Spring
  $ mvn install
  $ cd  src/main/webapp
  $ npm install
  $ tsc
```

## Run
In Angular2+Spring folder

```sh
  $ mvn spring-boot:run
```

Then, open your browser at http://localhost:8080/
